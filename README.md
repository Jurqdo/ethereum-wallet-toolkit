### üîë Features

- **Wallet Generation**: Create new wallets with BIP39/BIP44 standards.
- **Secret Storage**: Manage Web3 Secret Storage V3 keystores effectively.
- **Typed Data Signing**: Easily sign EIP-712 typed data with your wallet.
- **MCP Servers**: Generate wallets using Model Context Protocol servers.
- **Vanity Addresses**: Create personalized vanity Ethereum addresses.
